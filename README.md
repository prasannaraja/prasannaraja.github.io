# Prasanna Raja Portfolio

Modern portfolio website built with **React + Vite + Tailwind CSS**.

## Features

- Responsive modern dark UI
- Smooth scrolling navigation
- Glassmorphism cards
- Animated sections with Framer Motion
- Senior Flutter Developer branding
- GitHub / LinkedIn / Email CTA
- Ready for GitHub Pages or Vercel

## Run locally

```bash
npm install
npm run dev
```
